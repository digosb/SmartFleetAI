# 🚗 SmartFleet AI

O **SmartFleet AI** é um sistema de gestão de veículos desenvolvido em Python com o objetivo de evoluir para uma plataforma completa de gerenciamento de frotas, equipes e operações de campo.

O projeto nasceu como um sistema de controle de viagens e quilometragem, mas foi planejado para crescer de forma incremental, seguindo boas práticas de Engenharia de Software. Ao longo do desenvolvimento, novas funcionalidades serão adicionadas, como banco de dados, dashboards, relatórios, autenticação de usuários, integração com Power BI, Inteligência Artificial e uma versão web.

Mais do que um software, este repositório representa minha jornada de aprendizado em desenvolvimento de sistemas, arquitetura de software e Inteligência Artificial, aplicando tecnologias modernas para resolver problemas reais do dia a dia de empresas de infraestrutura.

## Objetivos do projeto

* Desenvolver um sistema profissional utilizando Python.
* Aplicar boas práticas de programação e arquitetura de software.
* Evoluir do armazenamento em Excel para banco de dados SQLite.
* Criar dashboards gerenciais com Power BI.
* Integrar recursos de Inteligência Artificial para automação e apoio à tomada de decisão.
* Disponibilizar futuramente uma versão web acessível pela internet.

## Tecnologias utilizadas

* Python
* Tkinter (e futuramente CustomTkinter)
* OpenPyXL
* SQLite
* Git
* GitHub
* Power BI (planejado)
* FastAPI (planejado)
* Inteligência Artificial (planejada)
